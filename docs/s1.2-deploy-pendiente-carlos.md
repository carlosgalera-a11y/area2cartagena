## 5 · Validar que askAi responde

```bash
firebase functions:shell
```

Dentro del shell:
```js
askAi({
  type: 'educational',
  prompt: 'Di OK en una palabra',
  systemPrompt: 'Eres un formador de MFyC',
}, { auth: { uid: 'test-uid' } })
```

Debe devolver `{ provider: 'deepseek', model: 'deepseek-chat', text: 'OK' (o similar), cached: false, ... }`.

Prueba también `type: 'clinical_case'` (debe ir a `provider: 'openrouter'`, modelo `google/gemini-2.5-flash-lite`).

## 6 · Logs

Tras una llamada real:
```bash
firebase functions:log --only askAi --lines 30
```

Deberías ver `askAi.call` con `{uid, type, provider, model, latencyMs, cacheHit, tokensIn, tokensOut, costEstimateEur, promptHash}`. **Nunca** el texto del prompt ni la respuesta.
