# Milrinone (Primacor®)
- Drug ID: 170
- Type: drug
- Entorno: adult

## Indicación: Pulmonary hypertension (nebulization)
- Categoría: Uncategorized
- Weight type: 2

  - STEP:   [Nebulization]  Repeat q6h PRN
      concentration: 1 mg/mL
      administration: If available, use a mesh nebulizer rather than a jet nebulizer as studies have shown that treatment with milrinone is probably more efficient when delivered with a mesh nebulizer.

Nebulize directly without dilution.

## Indicación: Cardiogenic shock
- Categoría: Uncategorized
- Weight type: 2

  - STEP: CrCl > 50mL/min
      concentration: 1 mg/mL
      administration: This drug should be adjusted in renal failure

There is no consensus in the literature on the optimal dosing of milrinone in renal failure.
Expert opinion for CrCl > 50mL/min:
Presentation: 1mg/mL (10mL vial)
1) Add 20mg (20mL) to 100mL of NaCl 0.9%
2) Do NOT remove volume
3) Start at the desired dose
4) Adjust according to clinical response from 0.375 to 0.5 to 0.75 mcg/kg/min every few hours (half-life 2-3 hours)

Manufacturer recommended adjustment :
ClCr 50mL/min/1.73m2 = 0.43 mcg/kg/min
ClCr 40mL/min/1.73m2 = 0.38 mcg/kg/min
ClCr 30mL/min/1.73m2 = 0.33 mcg/kg/min
ClCr 20mL/min/1.73m2 = 0.28 mcg/kg/min
ClCr 10mL/min/1.73m2 = 0.23 mcg/kg/min
ClCr 5mL/min/1.73m2 = 0.2 mcg/kg/min

Some sources recommend avoiding milrinone if ClCr <10mL/min/1.73m2
