# Methimazole PO (Tapazole®)
- Drug ID: 242
- Type: drug
- Entorno: adult

## Indicación: Thyroid storm
- Categoría: Uncategorized
- Weight type: 1

  - STEP: 
      administration: For patients who cannot tolerate the PO route, consider placing and administering via nasogastric tube. Alternatively, intrarectal administration is possible (see reference Nabil et al.). If this is still not possible, some sources describe how to prepare a solution for intravenous administration (see reference Hodak et al.).
      particularity: Some sources recommend a loading dose of 40mg. Subsequent doses 20mg q4-6h.

Wait at least 1 hour after administering methimazole before beginning iodine therapy.
